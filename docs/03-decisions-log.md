# Log des Décisions Techniques

## 20 juillet 2025

### Stack Frontend
**Décision** : React Native Expo  
**Alternatives** : React Native CLI, Flutter  
**Raison** : Simplicité déploiement, moins de config native, Expo services intégrés  

### Stack Backend  
**Décision** : Node.js + Express + MongoDB  
**Alternatives** : Python Flask, PHP Laravel  
**Raison** : JavaScript full-stack, MongoDB flexible pour prototyping  

### Hébergement
**Décision** : Heroku pour backend  
**Alternatives** : AWS, DigitalOcean, Railway  
**Raison** : Simplicité déploiement, pricing transparent, bon pour MVP  

### Images
**Décision** : Cloudinary  
**Alternatives** : AWS S3, local storage  
**Raison** : API simple, optimisation automatique, tier gratuit généreux  

### Géolocalisation
**Décision** : Google Maps API  
**Alternatives** : Mapbox, OpenStreetMap  
**Raison** : Documentation excellente, précision, intégration React Native
